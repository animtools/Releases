## [0.1.0] - 2026-02-22

### Added

- 初回リリース（Windows 64bit ポータブル版）
- 動画再生・シーク・音量調整
- タイル（サムネイル）の生成・表示
- ライブラリフォルダによる動画管理
- 多言語対応（日本語・英語ほか）

### Note

- ポータブル版は zip 解凍後の exe で利用できます。インストーラ版を同梱している場合は Setup 用 exe も含まれます。
