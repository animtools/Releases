# VideoTileKun - クイックスタートガイド

Windows 用ポータブル版の最短の使い方です。

---

## 1. ダウンロードと起動

1. [Releases](https://github.com/animtools/Releases/releases) を開く。
2. 最新リリースの **VideoTileKun-vX.Y.Z-win64.zip** をダウンロード。（X.Y.Z はバージョン番号。win64 は Windows 64bit 用のビルドを示す名前です。）
3. zip を解凍する（任意のフォルダでよい）。
4. 解凍したフォルダ内の **exe** をダブルクリックして起動。

※ 初回起動時、Windows のスマートスクリーンで「不明な発行元」と出ることがあります。詳細 → 実行 で起動できます（署名未対応のため）。

---

## 2. 起動後の流れ

1. 起動するとまず **プレーヤーウィンドウ** が開きます。
2. **ライブラリフォルダ** を指定してください（プレーヤーまたはメニューから設定）。生成したタイル（サムネイル）はこのフォルダにストックされます。
3. **ブラウザウィンドウ** を開くと、ライブラリの管理や、プレーヤーへ動画を読み込む操作ができます。

---

## 3. 動画でタイルを生成する

1. プレーヤーで動画を開く（ファイルを開く、またはブラウザから読み込み）。
2. **タイル生成** のボタン／メニューでサムネイルを生成する。
3. 生成されたタイルは **ライブラリフォルダ** にストックされ、ブラウザで一覧・管理できます。

---

## 4. ライブラリとプレーヤーの連携

- **ブラウザウィンドウ**: ライブラリフォルダ内の動画一覧の確認、プレーヤーへの読み込み、タイルの参照など。
- **プレーヤーウィンドウ**: 動画の再生・シーク、タイルの生成。タイルは自動でライブラリにストックされます。

---

## 5. 困ったとき

- **起動しない**: Windows 10/11 64bit か確認。別の PC やユーザーで試す。
- **タイルが生成されない**: 動画の形式やコーデックによっては失敗することがあります。別の動画で試す。
- **その他**: リポジトリ共通の [SUPPORT](../../SUPPORT.md) から不具合報告・質問ができます。
